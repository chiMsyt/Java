# Project 02: Simple Calculator

---

## Description

This project is a command-line calculator that performs basic arithmetic opertions: additin, subtraction, multiplication, and division.

---

## Key Skills

- Variables
- User input handlind using the Scanner class
- Conditional statements for operation selection

---

## Handling Input/Output

The program prompts the user for two numbers and an operator.
It then performs the selected operation and displays the result.
This project demonstrates how to handle user input and output effectively.

---

## Control Flow

The use of a switch statement allows for clean and efficient handling of different operations based on user input.
This project reinforces the importance of control flow in programming.
